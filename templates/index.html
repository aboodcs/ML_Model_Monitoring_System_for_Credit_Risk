Input Features

The model was trained on credit application data. Provide accurate values for best results — the pipeline applies the same scaler.pkl preprocessing at inference time.

person_age — Applicant's age
person_income — Annual income (USD)
person_emp_length — Employment length (years)
loan_amnt — Requested loan amount
loan_int_rate — Interest rate (%)
loan_percent_income — Loan as % of income
cb_person_cred_hist_length — Credit history (years)
person_home_ownership — RENT / OWN / MORTGAGE
loan_intent — Purpose of loan
loan_grade — Grade A–G
cb_person_default_on_file — Prior default Y/N
